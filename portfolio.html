Landing Page

This project required me to build a fully responsive landing page for a fake bank, working from a FIGMA design file. I created and organized the HTML and CSS, using flexbox for responsiveness and JavaScript for mobile menu interactivity.

Wordle Clone

For this project, I built a responsive clone of the popular word guessing game. I used CSS flexbox and animations to create a smooth gameplay experience. JavaScript is extensively used for game logic and interactivity through DOM manipulation, as well as for optional dark mode palette swap.

Job Listing

This was a small, responsive project that focused on using JavaScript and DOM manipulation to allow users to filter through job listings, allowing users to stack filters and individually remove them.

Etch-a-Sketch

This project allows the user to draw on a responsive virtual etch-a-sketch. I used JavaScript to edit CSS classes to allow for the functionality. I also used JavaScript to allow for different "resolutions" by dynamically generating different sized boxes to be automatically organized by CSS Grid.

Tip Calculator

This project was to create a responsive calculator to determine tip and total amounts on a bill, allowing for splitting between any number of guests. This project uses HTML and CSS to display a responsive visual. I made extensive use of JavaScript for mathematical functioinality, user input, and DOM manipulation.
